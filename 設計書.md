# あみだくじ飲みゲーム Web アプリ 設計書

---

## 1. アプリ概要

| 項目 | 内容 |
|---|---|
| アプリ名 | あみだくじ飲みゲーム |
| 目的 | 飲み会で誰が飲むかをあみだくじで決める |
| 対象人数 | 固定4人 |
| 対象端末 | **スマホ専用**（PC非考慮） |
| 技術スタック | HTML / CSS / JavaScript（単一ファイル、サーバー不要） |

```
                                    [あみだくじ画面（再生成）]
```

## 4. 機能仕様

### 4-1. あみだくじ生成ロジック

1. 縦線4本を均等配置
2. 各行で橋を配置（同じ行に橋が隣接しないよう制御）
3. 行数：10〜14行
4. 「飲み」をランダムな1カ所に配置、残り3カ所は「セーフ」

### 4-2. あみだくじ進行アニメーション

| フェーズ | 内容 |
|---|---|
| 開始 | 「スタート！」ボタンタップでアニメーション開始 |
| 進行 | 全4人が同時に上から下へパスを辿る |
| 色分け | 各プレイヤーの経路を別々の色でトレース |
| 完了 | 全員のゴールが確定 → 「飲み」の人がいればポップアップ表示 |

### 4-3. 結果表示

| 結果 | 表示 |
|---|---|
| セーフ | 経路をグリーンでハイライト |
| 飲み | 経路をレッドでハイライト → ポップアップ表示 |

## 5. UI/UX 仕様

### スマホレイアウト方針

- 幅基準：375px（iPhone SE相当）〜 430px（iPhone Pro Max）
- 縦スクロールなし（1画面に収める）
- タップターゲット：最低48px以上
- フォント：日本語フォント（system-ui）

### ボタン仕様

| ボタン | 配置 | サイズ |
|---|---|---|
| スタート！ | `position: fixed; bottom: 0` | 幅100%、高さ64px、文字24px |
| もう一度！ | `position: fixed; bottom: 0` | 幅100%、高さ64px、文字24px |

### カラーパレット

| 用途 | カラー |
|---|---|
| 背景 | ダークネイビー `#1a1a2e` |
| あみだ線（通常） | ホワイト `#ffffff` |
| セーフ経路 | ライトグリーン `#4ade80` |
| 飲み経路 | レッド `#ef4444` |
| セーフラベル | グリーン `#22c55e` |
| 飲みラベル | オレンジレッド `#f97316` |
| ポップアップ背景 | 半透明ブラック `rgba(0,0,0,0.85)` |
| ポップアップ文字 | ゴールド `#fbbf24` |
| スタートボタン | 鮮やかなオレンジ〜レッド `#f97316` |

### アニメーション

| アニメーション | 詳細 |
|---|---|
| 経路トレース | 線が上から下へ描画（全体で約2秒） |
| ポップアップ表示 | ズームイン + フェードイン |
| 「飲み」テキスト | バウンスアニメーション |

## 6. データ構造

```javascript
// プレイヤー情報
players = ["田中", "鈴木", "佐藤", "山田"]  // 4人固定

// あみだ構造
bridges = [
  { row: 0, col: 1 },  // col番目とcol+1番目の縦線を繋ぐ横線
  { row: 2, col: 0 },
  ...
]

// ゴール配置（シャッフルで「飲み」1つ）
goals = ["セーフ", "飲み", "セーフ", "セーフ"]

// 各プレイヤーの経路（アニメーション用）
paths[i] = [
  { x: col, y: row, direction: "down" | "left" | "right" }
]
```
